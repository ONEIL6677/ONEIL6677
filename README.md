## 🔐 DevSecOps Focus

Security isn't an afterthought — it's part of the pipeline. My DevSecOps practice
includes:

- **Static Application Security Testing (SAST)** scanning code for vulnerabilities with SonarQube and Snyk before it merges
- **Container Image Scanning** using Trivy to catch vulnerabilities in Docker images before deployment
- **Secrets Management** integrating HashiCorp Vault to eliminate hardcoded credentials
- **Infrastructure Security** scanning Terraform code with tools like Checkov and tfsec to catch misconfigurations early
- **Policy as Code** enforcing compliance and security guardrails automatically across environments

---

## 🤖 AIOps Interests

AIOps applies AI and machine learning to IT operations, helping teams move from
reactive to proactive operations. Areas I'm exploring include:

- **Anomaly Detection** identifying unusual patterns in metrics and logs before they become incidents
- **Predictive Scaling** using historical data to forecast resource demand and auto-scale proactively
- **Intelligent Alerting** reducing alert fatigue by correlating related alerts and surfacing root causes
- **Log Analysis with ML** using the Elastic Stack and ML models to detect patterns across large volumes of log data
- **ChatOps & LLM-Assisted Troubleshooting** using AI assistants to speed up incident diagnosis and runbook execution
